# Weekly Review

Generate a weekly review and next-week plan.

## Variables
(optional) $ARGUMENTS

---

## Instructions

### Inputs
- Last 7 daily briefs in `data/reports/`
- `context/current-data.md`
- Client status files in `clients/*/status/`

### Output
Create:

- `data/reports/weekly-review-YYYY-MM-DD.md`

Include:
- Wins shipped
- Metrics snapshot (even if approximate)
- What broke / root causes
- Templates created
- Case study progress
- Next week priorities (delivery/sales/content split)
